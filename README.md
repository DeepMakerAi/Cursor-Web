# Cursor Web 远程控制

通过Web界面远程控制Cursor的AI聊天功能 - 简化版

## 快速开始

### Windows 用户（推荐）

1. 下载最新版本的 `release.zip`
2. 解压到任意文件夹
3. 双击运行 `cursor-web.exe`
4. 打开浏览器访问 `http://localhost:3000`

### 开发者

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产模式
npm start

# 打包Windows可执行文件
npm run package
```

## 功能特性

- 🚀 一键启动，无需安装Node.js
- 🌐 Web界面远程控制Cursor
- 💬 实时消息同步
- 🔄 自动重连机制
- 📱 响应式设计

## 系统要求

- Windows 10/11 (64位)
- 或 Node.js 14.0.0+
